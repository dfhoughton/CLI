Download

A source tarball with documentation, ant build script, etc: cli-__VERSION__.tgz
For the most up-to-date code, clone the github repository: 
	git clone https://github.com/dfhoughton/CLI.git
Once you have the code, most things you may want to do can be accomplished with the build file.
If you wish to run the unit tests, you will need JUnit 4 on your classpath.
